# PirateStagram

This project is an Instagram clone made in Laravel and TailwindCSS.

**Instructions:**

1. Run Laravel with `./vendor/bin/sail up`
2. Run TailwindCSS with `npm run dev`
3. Open a browser and go to `localhost`
